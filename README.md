### 建立一個簡單的本機伺服器

可連線至：
- http://localhost:3000
- http://localhost:3000/getData
- http://localhost:3000/sum.html
- http://localhost:3000/myName

- `node`
- `express`
- `nodemon`
- `cookie-parser`
